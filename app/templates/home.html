Application purpose

Using the extraction tab after entering the product id from the ceneo website, we are able to download information about the product and its reviews.

How it works?

By specifying the product id in the extraction tab, the application checks whether the product exists. if so, it proceeds to extract the data. The data is saved in json format, so that it can be easily processed later. In the products tab we will see the tabs with the products that have been extracted. Each of the products has a tab with details and charts. In the details tab you will see a table with all the information about the product. going to the charts tab you will see two charts, pie chart and column chart.
